# Быстрый старт

## Установка и запуск (5 минут)

### 1. Установка зависимостей

```bash
cd "reflex-calculator"
python3 -m venv venv
./venv/bin/pip install -r requirements.txt
```

### 2. Запуск приложения

```bash
./venv/bin/streamlit run app.py --server.headless true
```

Откройте `http://localhost:8501`

### 3. Первое использование

1. **Выберите модель** в левой панели:
   - Model B (B2B2C) — рекомендуется для начала

2. **Выберите сценарий:**
   - Base — хороший стартовый сценарий

3. **Посмотрите результаты:**
   - KPI метрики вверху страницы
   - График Cash Flow — основной график
   - Блок "Целевая безубыточность" для автоподбора цены

4. **Измените параметры:**
   - Для ряда параметров переключайте режим `Слайдер` / `Ручной ввод`
   - Графики обновляются автоматически
   - Настройки сохраняются автоматически при изменении

### 4. Экспорт результатов

В конце страницы:
- Кнопка "ФЭМ с формулами (Excel)" — интерактивный файл
- Кнопка "Данные (Excel)" — снимок текущих расчетов
- Кнопка "Экспорт в JSON" — для данных

## Примеры задач

### Задача 1: Найти минимальную цену для безубыточности

1. Выберите Model B, сценарий Conservative
2. В блоке "Целевая безубыточность" задайте месяц выхода в плюс
3. Посмотрите "Минимальная цена аренды"
3. Сравните с текущей ценой аренды в параметрах

### Задача 2: Сравнить модели A и B

1. Выберите Model A, сценарий Base → запомните Net Cash Flow
2. Переключите на Model B, тот же сценарий → сравните Net Cash Flow
3. Какая модель прибыльнее?

### Задача 3: Оптимизация параметров

1. Откройте "Анализ чувствительности"
2. Найдите топ-3 параметра с максимальным влиянием
3. Настройте эти параметры для улучшения Cash Flow

### Задача 4: Проверить структуру revenue по месяцам

1. Откройте "График 2: Структура Revenue"
2. Выберите месяц через селектор
3. Сравните изменения структуры между месяцем 1 и последним месяцем горизонта

## Решение проблем

### Ошибка: "ModuleNotFoundError: No module named 'streamlit'"

**Решение:** Установите зависимости:
```bash
pip3 install -r requirements.txt
```

### Приложение не открывается в браузере

**Решение:** Откройте вручную:
```
http://localhost:8501
```

### Ошибка: "No such file or directory: config/defaults.json"

**Решение:** Убедитесь, что запускаете из директории `reflex-calculator`:
```bash
cd "reflex-calculator"
./venv/bin/streamlit run app.py --server.headless true
```

### Графики не отображаются

**Решение:** Обновите plotly:
```bash
./venv/bin/pip install --upgrade plotly
```
